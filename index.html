Help

The aim of the game is to find a 4-digit number in least possible guesses. The number consists of 4 distinct digits which starts with a non-zero.

For each guess made, a corresponding hint will be given which will help in making subsequent guesses.

Each digit which is present in the number with correct position, will be represented by "+" followed by its count.

Each digit which is present in the number but with incorrect position, will be represented by "-" followed by its count.

For example, if the 4-digit number is 5263 and if the guess made is 1234, then the hint will be "+1 -1". "+1" is for 2 which is a correct digit in correct position. "-1" is for 3 which is a correct digit in incorrect position.

If the guess is 5234, then the hint will be "+2 -1" as 5 and 2 are in correct positions and 3 is in incorrect position.

If the guess does not contain any digits present in the number (say 7890), then the hint will be "0".
